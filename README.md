# Agenda de contactos
Breve programa para agregar contactos y poder editar y eliminarlos en una base de datos

# Pasos para instalar el programa
- Hacer un git clone del proyecto
- Importar la base de datos
- Ejecutar xampp o wampp
- Disfrutar :)
